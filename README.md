# persistent

Это скрипт для автоматического создания persistent iso образов debian live и kali-linux live соответственно.
А также любых live дистребутивав семейства debian. Возможно сработает и над других live дистрибутивах.

## как использовать.
[скачайте скрипт]( https://github.com/33bc67db-c631-41e7-800a-8f96d841ebf0)

  запишите iso образ на флешку спомащью dd в терминале или спомощью руфус, также в режыме dd.
  А затем актевируйте скрипт.
  Скрипт спросит 'устройство' в это поле необходимо вписать путь к флешке например:

  ```bash
  /dev/sdx
  ```

а затем нажмите *Enter*.
после этих действий у вас будет готовая флешка с persistence.

---

## как запустить скрипт.

Втерминале linux введите:

```bash
`chmod +x persistence.sh`
```

—команда даёт скрипту права на выполнения

```bash
`./persistence.sh`
```

—запускает скрипт.
